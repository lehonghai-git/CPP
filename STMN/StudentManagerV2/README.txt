1. Thêm, Sửa xóa
	1.1. Thêm từ bàn phím: Nhập sinh viên thủ công từ bàn phím
	1.2 Thêm từ file: Nhập sinh viên từ file danh sách sẵn có (file phải được đặt cùng thư mục với file source code), các trường trong file cách nhau bằng dấu cách hoặc dấu tab
	1.3. Sửa: Sửa thông tin qua việc nhập số báo danh, nếu nhập sai sẽ không thể sửa
1.4. Xóa: Xóa sinh viên theo số báo danh, nếu nhập sai sbd cũng không thể sửa
1.5. Thoát
2. Tìm kiếm
	2.1. Theo tổng điểm: Nhập điểm tổng 3 môn để tìm kiếm
	2.2. Theo số báo danh: Nhập số báo danh để tìm kiếm (có phân biệt hoa thường)
	2.3. Theo tên: Nhập họ tên để tìm kiếm (có phân biệt hoa thường)
2.4. Theo điểm toán: Nhập điểm tóa để tìm kiếm
2.5. Theo điểm lý: tương tự bên trên
2.6. Theo điểm hóa: tương tự bên trên
2.7. Thoát
3. Sắp xếp
	3.1. Theo số báo danh: Sắp xếp theo thứ tự a-z, 0-9 của  số báo danh
	3.2. Theo Anphabe: sắp xếp theo a-z
	3.3. Theo tổng điểm: sắp xếp theo tổng điểm tăng dần
	3.4. Thoát
4. Xuất file
	Nhập tên file muốn xuất, file xuất ra nằm cùng vị trí với file source code
5. Thống kê
	Thống kê tất cả các sinh viên, thống kê theo các tiêu chí điểm.
6. Thoát
